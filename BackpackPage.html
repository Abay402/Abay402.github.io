Backpack Inventory System in Unity

I worked on this system recently as I wanted to refine my skills in Unity and try to make a system like this from scratch with as much flexibility as possible.

To make the system dynamic and flexible, it utilizes scriptable objects. I wanted everything to be easily edited in the editor so the size of the backpack and the size of each items are customizable and flexible.
